# DM Cheat Sheet – Ready-to-Run Encounters

## Encounter Format
Each encounter includes: Hook • Goal • Map/Space • Features • Enemies • Timer • Skill Challenges • Outcomes • Faction Shifts

---

## 1) Salt Stair – Tides & Names
### Hook
Shipwreck survivors beg for help as fog rolls in.

### Goal
Rescue survivors and secure passage.

### Map/Space
- **Layout**: Tiered spiral wharf with collapsing gantries
- **Terrain**: Difficult terrain near edges
- **Lighting**: Brine fog (lightly obscured, dampens fire)

### Features
- **Tide Windows**: Vault doors open during rounds 3-6
- **Brine Fog**: Light obscurement, fire damage halved

### Enemies
- **Barnacle Swarm** (Minions)
- **Smuggler Crew** (Talkative)
- **Brine Wraith Envoy** (Parleyable)

### Timer
**4 ticks** – High tide floods the lower tier.

### Skill Challenges
1. **Athletics** (DC 14): Secure ropes for safe passage
2. **Religion** (DC 12): Speak the names of the drowned
3. **Persuasion/Intimidation** (DC 13): Manage panicked crowd

### Outcomes
| Result | Effect |
|--------|--------|
| **Clean Success** | Survivors rescued + safe passage IOU |
| **Messy** | Injuries/gear loss |
| **Failure** | Losses + market distrust |

### Faction Shifts
- **Syndicate +1** if saved
- **Nazarick +1 Interest** if audit cache opened

## 2) Ashen Lyceum – The Lecture That Repeats
### Hook
A null-ink letter invites entry "by questions only."

### Goal
Retrieve the codex without breaking the parley rules.

### Map/Space
- **Layout**: Stacks and clockwork aisles
- **Dynamic**: Moving cover every other round

### Features
- **Chronomantic Lectern**: Replays last day
- **Screaming Grimoires**: Psychic pings on failed saves

### Enemies
- **Animated Curricula** (Constructs)
- **Dean's Echo** (Social duel)
- **Choir Saboteurs** (Skirmishers)

### Timer
**4 ticks** – Memory loop collapses to fire.

### Skill Challenges
1. **Arcana** (DC 15): Align the lectern
2. **Insight** (DC 14): Parse the Echo's arguments
3. **Stealth/Deception** (DC 13): Bypass saboteurs

### Outcomes
| Result | Effect |
|--------|--------|
| **Clean** | Codex + secret |
| **Messy** | Codex + debt |
| **Failure** | Fire + partial notes |

### Faction Shifts
- **Compact +1** if contained
- **Choir +1 Zeal** if exposed

## 10) Black Throne Parley – The Last Word
### Hook
Final terms before war or accord.

### Goal
Win concessions without breaking the hall's rules.

### Map/Space
- **Features**: Truth benches, safe-word sigils, oath-recording dais
- **Lighting**: Dim, with spotlights on speakers

### Features
- Lair focus candles (extinguish to suppress one lair action for 1 minute)
- Zone of Truth effect

### Enemies
None unless duel is invoked; then tailored champion appears

### Timer
**3 ticks** – Outside pressures force decisions.

### Skill Challenges
1. **Deception/Insight/History** (DC 16): Gain advantage in negotiations
2. **Performance** (DC 15): Sway the gallery
3. **Law (Intelligence)** (DC 17): Use legal technicalities

### Outcomes
| Result | Effect |
|--------|--------|
| **Clean** | Favorable accord reached |
| **Messy** | Mixed results, concessions made |
| **Failure** | Duel or siege begins |

### Faction Shifts
- All faction meters lock
- Epilogue branches based on choices made
