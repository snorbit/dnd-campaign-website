# Quest Log – Overlord: Ashes of the New World

## How to Use
- Fill one entry per session. Track the key choice, immediate outcome, clocks moved, and faction meter changes (0–5).
- Keep short but specific. Note promises made, debts incurred, NPCs affected, and new rumors created.

## Faction Meters (reference)
- **Nazarick**: Interest → Respect → Reliance → Control → Ownership.
- **Syndicate**: Trust → Routes → Leverage → Cartel → State.
- **Choir**: Purity → Zeal → Cells → Terror → Purge.
- **Compact (Free Cities)**: Diplomacy → Coin → Intelligence → Sanctions → Hegemony.
